QBullets tell you what a link will do before you click on it. QBullets are a collection of elegant, animated icons that attach to hypertext links to indicate their function. And they're free!

Using QBullets

For complete information about using QBullets, visit the Matterform Media Web site. You'll find information about the meaning of QBullets, how to use QBullets and more. Be sure to check out the DreamWeaver QBullet tools.
